¿Qué es CSS?

Selectores

Es todo un lenguaje de programación que nos permite dar estilo y forma a nuestro HTML (color, fuentes, tamaño, etc...).
Para poder activar el css dentro de un archivo HTML necsitamos la siguiente tag:

Style
Para poder aplicar el estilo necsitamos darle un nombre a la parte en HTML que se desse estilizar, para eso usamos el id, class, name
Font-family
Nos permite elejir la familia de fuentes que queramos tener en la pagina
Font-size
Nos permite elejir el tamaño de la fuente
Font-wight
Nos permite elejir el ancho de la fuente
Background-color
Nos permite elejir el color de fondo
Color
Nos permite elejis el color de la fuente
Text-align
Nos dice la alineación que tendra nuestro codigo de html
Text-decoration Nos permite elejir la decoración del texto
